# 影片分割程式

## 功能說明

此程式用於自動掃描指定資料夾中大於 200MB 的影片檔案，並將其分割成 2 個部分，以符合平台上傳限制。

## 特點

- 🔍 自動掃描資料夾及子資料夾中的影片
- 📏 識別大於 200MB 的影片檔案
- ✂️ 自動將影片從中間分割成 2 部分
- ⚡ 使用 ffmpeg 複製模式，分割速度快且不損失畫質
- 📁 **分割後的檔案留在原位置**
- 📦 **原始大檔案自動移動到 `原檔_超過200MB` 資料夾**

## 系統需求

- Python 3.6+
- **ffmpeg** (必須安裝)

### 安裝 ffmpeg

```bash
# 使用 winget 安裝
winget install ffmpeg

# 或從官網下載: https://ffmpeg.org/download.html
```

## 使用方式

### 1. 修改設定 (可選)

在 `video_splitter.py` 開頭可以修改以下設定：

```python
TARGET_FOLDER = r"D:\紫薇"  # 目標資料夾
SIZE_LIMIT_MB = 200         # 檔案大小限制 (MB)
```

### 2. 執行程式

```bash
python video_splitter.py
```

## 支援的影片格式

- MP4 (.mp4)
- AVI (.avi)
- MOV (.mov)
- MKV (.mkv)
- WMV (.wmv)
- FLV (.flv)
- WebM (.webm)
- M4V (.m4v)

## 注意事項

1. **分割後的檔案**會留在原始影片相同的目錄
2. **原始大檔案**會被移動到同目錄下的 `原檔_超過200MB` 資料夾
3. 如果分割後的檔案大小仍超過限制，可能需要調整分割數量
4. 使用複製模式分割，不會重新編碼，因此分割速度非常快

## 處理後的檔案結構

假設原始結構為：
```
D:\紫薇\
└── example_video.mp4 (250 MB)
```

處理後會變成：
```
D:\紫薇\
├── example_video_part1.mp4 (約 125 MB)
├── example_video_part2.mp4 (約 125 MB)
└── 原檔_超過200MB/
    └── example_video.mp4 (250 MB，原始檔案)
```
